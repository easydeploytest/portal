EasyDeploy Portal — developer dashboard

🚀 Deploy your app in 3 steps
1

Create repo from template

Go to github.com/easydeploytest/template and click Use this template → Create a new repository. Make sure to create it inside the easydeploytest org.

2

Configure app.yaml

Set name (unique, lowercase, ≥5 chars), team, and port to match your app's HTTP server. Commit your Dockerfile.

3

Push to main — that's it

CI triggers automatically. EasyDeploy builds your image, provisions Infisical, wires ArgoCD, and creates a Grafana dashboard. Your app appears in the table below when it's healthy.
